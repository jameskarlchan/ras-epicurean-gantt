Connect to Google Sheets

Your Google Sheet is the permanent home for the data. Edit there, regenerate here.

STEP 1. Click ⬇ CSV in the toolbar to download current task list with playbooks.
STEP 2. sheets.google.com → blank sheet → File → Import → Upload → drop CSV → Replace spreadsheet.
STEP 3. File → Share → Publish to web → Comma-separated values (.csv) → Publish → copy URL.
STEP 4. Paste the URL below. Click Connect. Generate now pulls live from the sheet.
